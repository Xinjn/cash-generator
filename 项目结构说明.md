# Cash Generator 项目结构说明

## 根目录文件

| 文件 | 作用 |
|------|------|
| `CashCouponGenerator.spec` | PyInstaller 打包配置文件，定义了入口脚本、隐藏导入模块（PyQt5、PIL）、资源文件路径、输出 App 名称和 bundle identifier |
| `requirements.txt` | Python 依赖清单，包含 Pillow（图像处理）、PyQt5（GUI 框架）、pyinstaller（打包工具） |
| `build_app.sh` | 一键打包脚本，自动清理旧构建缓存并执行 PyInstaller 命令生成 .app |
| `run.sh` | 一键运行脚本，直接 `python3 src/main.py` 启动测试，无需打包即可预览效果 |

## 构建输出目录

| 目录 | 作用 |
|------|------|
| `build/` | PyInstaller 构建临时文件，包含中间编译产物（.pyz、.pkg 等），可安全删除 |
| `dist/` | PyInstaller 打包输出目录，包含最终生成的 `CashCouponGenerator.app` 和未打包的可执行文件 |

## src/ 源码目录

| 文件 | 作用 |
|------|------|
| `__init__.py` | Python 包标识文件（空文件），使 src 目录成为一个可导入的 Python 包 |
| `main.py` | 应用程序入口点，自动检测命令行参数决定运行 CLI 模式（命令行生成）还是 GUI 模式（图形界面） |
| `app.py` | PyQt5 GUI 界面实现，包含窗口布局（QFormLayout）、控件样式（QSS）、用户交互逻辑（生成/错误提示） |
| `config.py` | 配置文件，定义 6 个国家（MY/TH/ID/PH/SG/VN）的货币符号、位置、分隔符等参数，以及 3 套模板样式（lazcash/shopee_coins/tokopedia_deals）的颜色、圆角、文字位置 |
| `generate.py` | 新版图像生成引擎，实现 9-patch 模板自适应缩放（保持四角圆角不变形）、货币格式化、字体加载与回退机制（支持 ₫฿₱ 等特殊字符）、自适应文字排版 |
| `generator.py` | 旧版图像生成器，功能与 generate.py 类似但实现方式不同 |
| `gui.py` | 旧版 GUI 实现（基于 Tkinter），因 Python 3.8 的 Tkinter 8.5.9 在 macOS 上存在渲染问题，已被 app.py（PyQt5）替代 |

## assets/ 资源目录

| 目录 | 作用 |
|------|------|
| `fonts/` | 字体文件，包含 EuclidCircularA-Bold.otf、EuclidCircularA-Medium.otf、Roboto-Black.ttf |
| `logos/` | 各国 logo 图片（66x66 RGBA 格式），包含 logo_MY.png、logo_TH.png、logo_ID.png、logo_PH.png、logo_SG.png、logo_VN.png |
| `templates/` | 券模板底图，包含 template_base.png（260x260 RGBA） |
| `output/` | 生成的券图输出目录 |
